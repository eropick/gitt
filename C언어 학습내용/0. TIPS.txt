뒤가 5로 끝나는 숫자의 제곱

ex) 25 ,75

25제곱 =>  2*3//5*5 = 625

75제곱 => 7*8//5*5 = 5625

n5제곱 => n*(n+1)//25

앞자리 숫자와 앞자리 숫자에 1을 더한 수를 곱하고 뒤에 25를 붙임. 

vim

입력모드
i : 커서 앞에서 입력
o: 아래 새 줄 삽입 후 입력
shift + i : 커서 라인 맨앞에서 시작
shift + a: 커서 라인 맨 뒤에서 시작 
shift + g : 문장 가장 끝으로 이동
숫자 + gg : 숫자 라인으로 이동
w : 단어단위로 이동
